Detailed Description

This struct provides an easy way for the user to set the sample time of a channel. All the options when selected will have an additional 12 cycles added to them in the hardware. So 3_Cycles is selected by the user, then total cycles taken to complete one conversion of that channel will be 3+12 = 15 cycles. Is used in _ADC_Channel struct.
