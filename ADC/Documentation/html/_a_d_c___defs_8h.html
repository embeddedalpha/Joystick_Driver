typedef struct _ADC_Channel_Sample_Time_ _ADC_Channel_Sample_Time_

Sample time configuration for an ADC channel.

This struct provides an easy way to set the sample time of an ADC channel. All options selected will have an additional 12 cycles added to them in hardware. For example, if _3_Cycles is selected, the total cycles taken to complete one conversion of that channel will be 3 + 12 = 15 cycles.

This struct is used in the _ADC_Channel_ struct.
